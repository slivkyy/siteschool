Во время создания сайта я много раз использовал тег "div"

Сейчас я вам расскажу про этот тег!

есть два вида тега "div":

есть div klass="название", а так же div id="название"

различаются они только написаием в CSS. div class пишется: .название; а div id пишется: #название
